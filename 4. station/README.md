# Station (두 선분 사이의 최단 거리)

3차원 공간 상에 존재하는 두 개의 선분(Segment) 사이의 **최단 거리**를 구하는 알고리즘입니다.

## 🚀 문제 정의

- 3차원 좌표계에서 선분 AB와 선분 CD가 주어집니다.
- 두 선분 위의 점 $P \in AB$, $Q \in CD$에 대하여, 거리 $dist(P, Q)$의 최솟값을 구해야 합니다.
- 각 선분은 매개변수 $t \in [0, 1]$를 이용하여 다음과 같이 표현됩니다:
  - $P(t) = (1-t)A + tB$
  - $Q(s) = (1-s)C + sD$

## 🛠 알고리즘: 중첩 삼분 탐색 (Nested Ternary Search)

두 선분 사이의 거리 함수는 **볼록 함수(Convex function)**의 성질을 가집니다. 따라서 극값을 찾기 위해 **삼분 탐색**을 활용할 수 있습니다.

1. **내부 삼분 탐색 (`find_shortest`)**:
   - 선분 AB 위의 특정 점 $P(t)$가 고정되었을 때, 선분 CD 위의 점 $Q(s)$와의 거리가 최소가 되는 $s$를 삼분 탐색으로 찾습니다.
2. **외부 삼분 탐색 (`solve`)**:
   - 내부 삼분 탐색을 통해 얻은 `f(t) = min_dist(P(t), CD)` 역시 $t$에 대해 볼록한 함수입니다.
   - 따라서 $t$를 범위 $[0, 1]$에서 삼분 탐색하여 전체 최솟값을 찾습니다.
3. **정밀도 확보**:
   - 약 100회의 반복(Iteration)을 통해 충분한 수치적 정밀도를 확보합니다.

## ⏱ 시간 복잡도

- **O(log(1/ε)²)**: 여기서 ε은 요구되는 정밀도입니다. 중첩된 삼분 탐색을 수행하므로 로그의 제곱에 비례하는 시간이 소요됩니다.
